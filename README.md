## Build from source
Run `make_exe.cmd` or
```
set TCL_LIBRARY=P:\Program Files (x86)\Python36\tcl\tcl8.6
set TK_LIBRARY=P:\Program Files (x86)\Python36\tcl\tk8.6
python setup.py build
```
